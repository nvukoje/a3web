GIS Databases

Data Engineering and Manipulation is Key to a Fine Geospatial Product

Through the use of Structured Query Language (SQL) and Relational Database Management Systems (DBMS), our consultancy provides the best geographic information to our clients. We are familiar with multiple systems besides Microsoft Access including PostGreSQL, Oracle RDBMS, Microsoft SQL Server and My SQL. These databases contain schemas and tables that are filled to the brim with geographic information. Using queries, these databases pull useful information for insightful decisions.

The tables largely consist of descriptive values of varying data types that impact the space (and cost) of the database. These data types consist of descriptive objects (text), metric values of height or distance (integers), percentages (float) or geographic coordinates (double).

Remote Sensing and Image Analysis

Collect Imagery and Information from the Electro-Magnetic Spectrum

Remote sensing and imagery are utilized to cover large swaths of land through the use of satellites or aerial photography. These images contain pixelated values based on different bandwidths within the electro magnetic spectrum. Depending on a object and its exposure to light, reflectance values can be used to calculate qualtities of said object. This can include forests both coniferous and deciduous, arable lands with crops emerging, water bodies with sediment and algae as well as urban areas.

An image in Southern Ontario can be shown with different RGB bands and their reflectance values. These images can have different RGB band combinations that create 'False Colour Composite' images in contrast to the 'True Colour Composite' image. Images can also be filtered to highlight different features in the land, such as urban areas. These land features are sharpened to be more visible for the client.
